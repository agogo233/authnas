# Security Groups | 安全组

安全组用于控制用户对应用和资源的访问权限。

## 创建安全组

在管理面板的 Groups 页面创建新的安全组。

## 组设置

| 设置 | 说明 |
|------|------|
| MFA Required | 组成员必须使用多因素认证 |
| Auto Assign | 新邀请用户自动加入该组 |

## 使用场景

**OIDC 应用**
- 设置应用允许访问的用户组
- 用户组信息可通过 `groups` scope 传递给应用

**ProxyAuth 域名**
- 设置域名允许访问的用户组
- 用户组信息通过 `Remote-Groups` 头传递给后端应用
