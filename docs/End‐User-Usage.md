# User Guide | 用户指南

## 登录 | Login

访问应用后会被重定向到 AuthNas 登录页面。

支持登录方式：
- 用户名 + 密码
- Passkey（Face ID、Windows Hello 等）
- "记住我" 选项可使会话持续长达一年

## 多因素认证 | MFA

当账户或应用要求 MFA 时，用户需使用 TOTP 应用或 Passkey 完成二次验证。

## 注册 | Sign Up

如果管理员开启了自注册功能，登录页会显示"注册"选项。

## 邀请注册 | Accept Invitation

通过邀请链接注册时，部分字段可能已预填充。

## Passkeys

Passkey 提供更安全的无密码登录体验。

**添加新 Passkey：**
1. 登录后访问"安全"设置页面
2. 点击"注册 Passkey"
3. 按设备提示完成操作

## 忘记密码 | Forgot Password

在登录页点击"忘记密码"，输入用户名或邮箱收取重置链接。

## 个人设置 | Profile Settings

登录后访问个人设置页面，可管理：
- 头像和名称
- 邮箱地址
- 密码
- Passkeys
- 安全设置
